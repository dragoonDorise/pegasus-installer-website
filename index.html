Welcome to Pegasus installer!(Beta)

An almost one touch configurator for Android Handelds

This guide will help you install, configure Pegasus Frontend and scrap your roms artwork all without leaving your Android Handheld.

Enable installing APK,in your Android device on Settings → Security → Unknown Sources so we can install Pegasus

Download and install Termux on your Android Device from this link so we can run all the automation scripts.

Copy and paste the folowing script onto Termux and press the A button / Enter

/bin/bash -c "$(curl -fsSL https://bit.ly/3ntlzL8)"
